# mxsscanner
A massive XSS detector.

Modes of use:

For a single URL analysis:
*ruby mxsscaner.rb --url* or *ruby mxsscaner.rb -u*

For a massive URLs file:
*ruby mxsscaner.rb --load [name_of_file]* or *ruby mxsscaner.rb -l [name_of_file]*
